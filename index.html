NVX Preview Dashboard is a third-party software tool built on the Crestron DM NVX® device API. It gives audio-visual professionals greater visibility over their DM NVX network — showing live stream previews, device status, and signal routing information in a single browser-based interface. It is not an official Crestron product.

You need an existing Crestron DM NVX installation — at least one NVX encoder on your local network. The dashboard software itself runs on any Windows PC on the same network. No additional hardware is required.

The dashboard communicates directly with your NVX devices over your local network. However, a periodic internet connection is required — the software performs a license validation check approximately once a week to confirm your licence is active. An internet connection is also needed for initial activation and software updates.
